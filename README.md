## 简介

「智能饮水助手」是一款基于浏览器端物理模拟的交互工具。您可以通过调整容器参数（形状、尺寸、材料）、液体类型和环境条件，实时模拟杯中液体的温度场变化过程，并通过 3D 可视化直观查看热力分布，科学计算最佳饮用时机。

无需安装任何软件，打开浏览器即可运行。

## 核心原理

### 热传导物理引擎

应用的核心是一个基于有限差分法的热传导模拟器，求解圆柱坐标系下的二维热传导方程：

```
∂T/∂t = α · (∂²T/∂r² + ∂²T/∂h²)
```

其中：
- **T** — 温度（°C）
- **t** — 时间
- **α** — 热扩散率（m²/s），取决于液体类型
- **r** — 径向坐标（杯壁方向）
- **h** — 轴向坐标（杯高方向）

### 物理引擎实现

文件：[physics-engine.js](file:///workspace/physics-engine.js)

1. **网格离散化**：将液体区域划分为 20×24 的网格，每个网格点记录温度值
2. **时间步进**：以 100 个时间步（共计 600 秒）模拟温度变化过程
3. **散热机制**：
   - 杯壁散热 — 受杯壁材料导热系数影响
   - 液面蒸发散热 — 表层液体散热更快
   - 内部热扩散 — 相邻网格间的热量传递
4. **杯壁材料影响**：5 种材料各有不同的保温系数，直接影响散热速率
5. **探针查询**：支持任意位置任意时刻的温度查询

### 影响散热的因素

| 因素 | 影响方式 |
|------|---------|
| **杯壁材料** | 导热系数越高，边缘散热越快 |
| **液体类型** | 热扩散率不同（茶水、咖啡、果汁、白开水） |
| **杯体尺寸** | 直径和高度影响体积与表面积比 |
| **初始温度** | 与环境温差越大，散热越快 |
| **环境温度** | 决定了最终平衡温度 |

## 操作指南

### 1. 参数设置（左侧面板）

| 参数 | 说明 |
|------|------|
| **杯子形状** | 圆柱形、锥形、方杯 |
| **杯壁材料** | 陶瓷、玻璃、不锈钢、塑料、硅胶 |
| **直径/高度/壁厚** | 通过滑块或数字输入调整 |
| **初始温度** | 饮品刚倒入时的温度 |
| **环境温度** | 当前室温 |
| **目标温度范围** | 适合饮用的温度区间 |
| **液体类型** | 茶水、咖啡、果汁、白开水 |

### 2. 开始模拟

点击 **「开始模拟计算」** 按钮：
- 物理引擎执行 100 步热传导计算
- 生成 3D 液体模型并着色（红色 = 高温，蓝色 = 低温）
- 右侧显示 AI 饮水策略分析
- 底部时间轴显示当前时刻

### 3. 查看热力分布

- **拖拽旋转**：鼠标拖拽 3D 模型，从任意角度观察温度分布
- **滚轮缩放**：拉近/拉远观察细节
- **时间轴**：拖动滑块查看不同时刻的温度场变化
- **播放按钮** ▶ 自动播放温度变化过程

### 4. 探针功能

探针可查询液体中任意位置的温度：

- **点击模型**：在 3D 液体模型上点击，放置探针
- **当前温度**：探针位置实时显示当前温度
- **温度曲线**：探针面板内自动绘制该点的温度-时间变化趋势图
- **添加至图表**：将探针数据添加到右侧温度总图表，可同时对比多个位置
- **坐标输入**：在探针面板中输入 X、Y、Z 坐标直接定位

### 5. 结果解读

- **温度变化曲线**（右侧面板）：展示探针位置的温度随时间变化
- **AI 饮水策略分析**：根据模拟结果推荐最佳饮用阶段和各阶段的饮用位置
- **物理参数**：显示当前液体和杯壁材料的物理属性

## 设计说明

### 技术栈

| 技术 | 用途 |
|------|------|
| **Three.js** | 3D 场景渲染、轨道控制器 |
| **Chart.js** | 温度-时间曲线图表 |
| **原生 JavaScript** | 物理引擎、应用逻辑 |
| **CSS3** | 响应式布局、主题系统 |

### 视觉设计

设计风格参考 Airbnb 设计语言：

- **配色**：暖色主调（珊瑚红 #FF5A5F + 青绿 #00A699）
- **字体**：SF Pro Display / -apple-system，干净现代
- **圆角**：统一的 12px 圆角系统
- **阴影**：柔和阴影营造层次感
- **浅色主题**：白色背景 + 浅灰卡片，内容突出

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  Header (logo + 状态栏)                              │
├──────────┬───────────────────────┬──────────────────┤
│  左侧面板 │      3D 场景区域      │    右侧面板       │
│  容器参数 │  (热力图 + 探针)      │  温度曲线         │
│  温度设置 │                      │  饮水策略         │
│  液体类型 │  时间轴 + 播放按钮    │  物理参数         │
│  操作按钮 │                      │                  │
└──────────┴───────────────────────┴──────────────────┘
```

### 响应式适配

- **桌面端（>1024px）**：三栏布局，完整功能
- **平板端（768-1024px）**：调整面板宽度
- **移动端（<768px）**：单栏纵向布局
